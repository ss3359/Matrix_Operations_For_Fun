# Matrix_Operations_For_Fun
This is a C++ and Java Project writing up algorithms for familiar matrix operations and properties of matrices. I hope you enjoy this project. 

I have updated this project by adding in Row Reduction in Echelon Form. Also, as of today, 1/3/2025, I have updated this code which includes the algorithm for the Gram-Schmidt process. I have also used Java to include The QR factorization and algorithm method to calculate the eigenvalues of a matrix. 

I hope you enjoy it!
